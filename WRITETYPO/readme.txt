C’est une base pour créer des lettres typographique sur le modèle de s lettres en plomb à imprimer en 3D.

Créé par GAU (gau@unseen.is / @gfcdesign / gfcdesign.com) dans le cadre d’un projet mené au Hackerspace du golfe du Lyon, LE BIB (lebib.org).

Il est fait usage du module write.scad créé par Harlan Martin
